## <a name="whats-translated"></a>번역된 내용

앱은 언어 기본 설정을 사용하여 메뉴, 레이블 텍스트 및 시스템 메시지와 같은 요소를 기본 언어로 표시합니다.

대부분의 기능은 데이터를 기반으로 하기 때문에 일부 항목은 원래 언어로 유지됩니다. 예:

- 엔터티 및 세그먼트를 포함한 데이터
- 환경 이름
